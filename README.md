# blacksite-site

Premium one-page public excerpt for BLACKSITE, a private, scope-guarded security research framework by Redacted.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Deploy

This is a static-first Next.js 15 App Router project with no backend, database, CMS, or auth. It is ready for Vercel: import the repo, keep the default Next.js settings, and deploy.

## Design Notes

- Restrained dossier aesthetic: near-black field, muted red accents, monospaced metadata, redaction bars, and subtle signal motion.
- Content is intentionally sparse and public-facing.
- Public excerpt only; no operational details, target data, findings, secrets, or private methodology are included.
